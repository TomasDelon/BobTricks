# Organisation des assets de presentation

Ce dossier contient uniquement les assets utilises par `doc/presentation/presentation_shell.html`.

- `images/` : captures, pictogrammes et schemas raster (`.png`).
- `videos/` : demonstrations integrees aux slides (`.mp4`).
- `diagrams/` : sources et exports de diagrammes (`.mmd`, `.svg`).
- `notes/` : notes orales exportees pour la presentation (`notes.js`, `notes.json`).
