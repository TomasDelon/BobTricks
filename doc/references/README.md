# References

## Contenido

- `deep-research-report.md`: sintesis larga de literatura tecnica para equilibrio, IK, control fisico y validacion.
- `papers/`: biblioteca de papers y capturas HTML guardadas localmente.
- `assets/`: imagenes de apoyo, por ahora nombres de nodos del stickman.

## Uso

- Leer primero `deep-research-report.md` para orientacion.
- Ir luego a `papers/` solo cuando haga falta profundizar en un subtema concreto.
- No asumir que todos los papers tienen el mismo peso; varios son historicos o exploratorios.

## Temas fuertes de esta biblioteca

- locomocion bipedal
- capture point y push recovery
- IK y reconstruccion
- active ragdoll y PD estable
- biomecanica de marcha, carrera, salto y aterrizaje
